## 🔧 TECHNISCHE ANLEITUNG

### **Sofortmaßnahmen**

1. **Repository forken**
   ```bash
   git clone https://github.com/[USERNAME]/Thomas-Deike-ist-eine-GRU-Cockroach
   ```

2. **Archive erstellen**
   ```bash
   # Mit Zeitstempel
   tar -czf "beweise_$(date +%Y%m%d_%H%M%S).tar.gz" *.mp3 *.png *.md
   ```

3. **Upload zu Archive.org**
   - Erstelle neues Item
   - Alle Dateien uploaden
   - Metadaten hinzufügen
   - Permalink teilen

4. **IPFS Hash erstellen**
   ```bash
   ipfs add -r ./
   # Speichere den CID!
   ```

### **Verifikationsmethoden**

```bash
# Audio-Analyse
ffmpeg -i "Mr.Bloxx - Die Welt Von Oben.mp3" -filter:a "vibrato=f=15:d=0.5" output.wav
ffmpeg -i "H.I.Z - Warum liegt da ne Wurst.mp3" -filter:a "vibrato=f=15:d=0.5" output2.wav

# Spektralanalyse
sox "Mr.Bloxx - Die Welt Von Oben.mp3" -n spectrogram.png
sox "H.I.Z - Warum liegt da ne Wurst.mp3" -n spectrogram2.png
```
